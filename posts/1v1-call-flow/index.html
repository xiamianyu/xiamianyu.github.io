1v1通话流程

泛娱乐音视频社交App的1v1通话流程梳理。

通话操作

  • 拨打方：dial, cancel
  • 接听方：accept, reject
  • 通话双方：hang

通话状态

  • IDLE
  • WAITING
  • CONNECTED

通话状态机

当前状态 触发事件 目标状态
IDLE dial WAITING
WAITING cancel IDLE
WAITING reject IDLE
WAITING timeout(sys) IDLE
WAITING accept CONNECTED
CONNECTED hang IDLE

API

  • /call/dial：拨打
  • /call/cancel：取消
  • /call/accept：接听
  • /call/reject：拒绝
  • /call/hang：挂断
  • /call/hb：发送心跳
  • /call/start：开始通话

IM

  • RING：来电
  • RING_ACCEPT：来电被接听
  • RING_CANCEL：来电被取消
  • RING_REJECT：来电被拒
  • RING_TIMEOUT：来电超时
  • HANG：通话被挂断
