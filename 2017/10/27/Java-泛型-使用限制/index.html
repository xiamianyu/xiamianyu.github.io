Java 泛型-使用限制

为了有效使用泛型，我们必须考虑以下的限制：

  • 不能使用基本类型来初始化泛型类型
  • 不能创建类型参数的实例
  • 不能声明数据类型为类型参数的静态域
  • 不能在参数化类型上使用 Casts 或 instance of
  • 不能创建参数化类型的数组
  • 不能创建、捕获或抛出参数化类型的对象
  • 一个类不能有多个在类型擦除后具有相同签名的重载方法
