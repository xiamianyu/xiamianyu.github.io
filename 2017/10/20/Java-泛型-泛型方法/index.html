Java 泛型-泛型方法

2017-10-20 | Java

泛型方法（Generic Method）是引入类型参数（Type Parameter）的方法。它类似声明泛型类型，但是类型参数的作用域仅限于自己。Java 允许静态和非静态的泛型方法，泛型构造方法也是。

泛型方法的语法：在尖括号内包含一个类型参数列表，出现在方法的返回值前面。
