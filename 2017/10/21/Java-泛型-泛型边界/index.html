Java 泛型-泛型边界

在泛型使用过程中，有时候我们希望可以限制类型参数（Type Argument）的类型，这时就会用到泛型边界。使用泛型边界很简单，在定义泛型类型或泛型方法时，使用 extends 关键字即可。

泛型类型边界
