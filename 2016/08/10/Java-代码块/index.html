Java 代码块

Posted on 2016-08-10   |   In Java   |     |   visitors

Java 中的代码块用{}修饰，分为三种：静态代码块、构造代码块和局部代码块。

静态代码块

类加载的时候被执行，而且只执行一次，优先于Main()方法执行，作用于类，对类进行初始化。

构造代码块

优先于构造方法执行，作用于对象，对多个构造方法中相同的操作做统一处理。

局部代码块

跟普通代码执行顺序一样，作用于对象，限定变量生命周期，及早释放，提高内存利用率。

结论
