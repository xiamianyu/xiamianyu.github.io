Java 反射

Posted on 2016-07-31   |   In Java   |     |   visitors

概念

反射是指在程序运行时获取类的属性、方法、接口等信息，并且可以创建类的实例，改变其属性。

原理

一个 Java 源文件被编译后生成 class 文件，这个文件中包含了该类的属性、方法、接口等信息，程序运行时它会被 ClassLoader 加载到 JVM 中。当类被加载后，JVM 会在内存中自动产生一个 Class 对象。我们通过 new 的形式创建对象实际上就是通过这些 Class 来创建，只是这个过程对于我们是不透明的而已。

提高反射速度

Java 中 AccessibleObject 类用于控制是否进行访问检查，它的子类有 Constructor、Field 和 Method，即是当我们获取到 Constructor、Field 或 Method 的对象时，通过 setAccessible(boolean flag) 来设置是否要进行访问检查，flag 为 true，取消访问检查，反之进行访问检查。通过取消访问检查可提升反射速度。
