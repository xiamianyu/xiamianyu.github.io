Chrome 插件推荐

  • chrome
  • 插件

前段时间stormzhang大哥在他的公众号AndroidDeveloper中, 发表了一篇文章「吐血推荐珍藏的Chrome插件」，特别棒！当时就立马下载去了。今天跟大家分享我觉得比较好用的插件，其中一些是从stormzhang大哥那里引用过来的，哈哈。

Postman

Postman是API调试的神器，基本功能如下：

  • 支持GET、POST、PUT、HEAD、DELETE等请求
  • 对请求返回的数据，支持JSON、XML、HTML、Text查看格式
  • 类似于浏览器，可以浏览和清空历史请求记录
  • 支持以文件夹的形式将API管理起来并保存到本地或云端，也可以将该文件夹导出来

我个人的使用习惯是，将项目的所有API，在Postman中用文件夹管理起来，以后测试和查看接口，都不用打开接口文档，直接打开Postman，点击接口测试即可，大大提高工作效率。

JSONView

直接在浏览器上对接API，一般返回的JSON数据没有格式化，看起来很乱，而JSONView可以将它们自动格式化好。

JSON-handle

对于嵌套层级很深的JSON数据，如果想单独查看或复制某个字段下的数据就会很猥琐，而JSON-handle可以很好地处理这猥琐的事，让你倍儿爽。

Octotree

在Github上查看源码是一件很痛苦的事，要clone下来，导入IDE或者stublime text才能更好的看清项目的结构，而Octotree会在Chrome的侧边栏展示项目的结构，方面你阅读源码。

Isometric Contributions

像盖楼一样展示你Github上的commit信息，很好玩！

Avatars for Github

在GitHub主页动态中显示用户头像

ChromeADB

通过USB或WiFi连接手机后，可以清理应用数据、卸载应用、暂停应用以及查看应用内存使用情况等

Vysor

让你在电脑上查看和操作Android手机

One-Click Extensions Manager

支持一键管理所有扩展，包括一键禁用所有扩展、一键开启原有扩展、扩展的逐个禁用和开启、右击扩展等功能

OneTab

一键保存打开的Tab，需要的时候还原。节省内存消耗，还你一个干净的Tab。

Pocket

好文章，来不及读，那就放入Pocket吧，等你有空再回来看看。

AdBlock

屏蔽互联网广告，你值得拥有！

Bookmark Manager

由Google打造的优雅书签管理器

Speed Dial2

有快速访问您最常访问网页的功能，书签和浏览历史记录的终极新标签页替代品

Chrono

Chrono接管你在Chrome中的所有下载，你的所有下载管理工作都在浏览器中完成，而不需要安装另外的程序。Chrono与Chrome浏览器紧密地整合在一起，添加了对浏览器菜单、工具栏和快捷键的支持。

Chrome Gestures

支持鼠标手势,超级拖曳,滚轮手势,摇杆手势,平滑滚动,标签页列表等

Firebug

Chrome网页调试工具

Reader View

提供与Safari阅读模式功能一致的插件，浏览文章页时候可进入友好的阅读模式，并自定义阅读功能

Draw.io Desktop

离线免费的画图工具，支持流程图、UML、IOS/Android原型设计等

Gliffy Diagrams

同上，也是一款非常好的画图工具

Page Ruler

网页尺寸测量工具

Google Keep

快速记录您的所思所想

Minimalist Markdown Editor

离线免费的 Mini Markdown 编辑器

Pomodoro Timer

番茄土豆是番茄工作法和 To-do List 工具，它可以帮助你计划、管理、执行、记录工作。现在就开始使用番茄土豆，完成了不起的工作！
