Charles Response 中文乱码

Posted on 2016-05-07   |   In 工具

使用 Charles Http 抓包时，Response 中文显示乱码，解决方案如下：

找到 Info.plist

Applications -> Charles -> Contents -> Info.plist

修改 Info.plist

打开 Info.plist，找到 -Xmx256m，添加 -Dfile.encoding=UTF-8 ，重启 Charles 即可

参考
http://blog.csdn.net/huanghr_1/article/details/39735335
